# Non-EEG Epilepsy Detection System
A multi-sensor wearable system designed to detect epileptic seizures without using EEG.  
This project uses **IMU-based motion analysis (MPU6050)**, **heart-rate & SpO2 monitoring (MAX30102)**, and an **ESP32 microcontroller** to detect seizure patterns in real-time.
